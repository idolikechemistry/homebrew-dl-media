# homebrew-dl-media

## 簡介

homebrew-dl-media是一個用於安裝/解除安裝[dl-media](https://github.com/idolikechemistry/dl-media)的Homebrew Cask。

此Homebrew Cask不包含測試版，若希望使用測試版，請從[https://github.com/idolikechemistry/dl-media/releases/](https://github.com/idolikechemistry/dl-media/releases/)下載帶有「pre-release」標記的版本。
